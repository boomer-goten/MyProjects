Smart_Calculator_v1.0

Приложение написано с использованием библиотеки программного интерфейса QT. Реализация на языке программирования С++ с использованием объектно-ориентированного подхода расширенная версия обычного калькулятора, который можно найти в стандартных приложениях каждой операционной системы. Помимо базовых арифметических операций, как плюс-минус и умножить-поделить, реализован калькулятор c возможностью вычисления арифметических выражений с учетом приоритетов, а так же некоторыми математическими функциями (синус, косинус, логарифм и т.д.). Помимо вычисления выражений калькулятор поддерживает использование переменной x и построение графика соответствующей функции. В качестве дополнительных улучшений рассмотрен кредитный и депозитный калькулятор. На вход программы могут подаваться как целые числа, так и вещественные числа, записанные через точку. У пользователя есть 

Сборка программы настроена с помощью Makefile со стандартным набором целей для GNU-программ: all, install, uninstall, clean, dvi, dist, test, gcov_report.

Режим кредитного калькулятора:

  • Вход: общая сумма кредита, срок, процентная ставка, тип (аннуитетный, дифференцированный)
  • Выход: ежемесячный платеж, переплата по кредиту, общая выплата

Режим депозитного калькулятора:

  • Вход: сумма вклада, срок размещения, процентная ставка, налоговая ставка, периодичность выплат, капитализация процентов, список пополнений, список частичных снятий
  • Выход: начисленные проценты, сумма налога, сумма на вкладе к концу срока

Начало работы

Чтобы начать пользоваться калькулятором, его необходимо установить с помощью команды make install. Эта команда установит программу в папку build в папке с исходниками проекта.

Удаление

Удалить пользовательскую программу. Вы можете использовать команду make uninstall
